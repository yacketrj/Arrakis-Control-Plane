# Dune: Awakening Incident Records

This folder contains case-specific incident records for Dune: Awakening self-hosted server support work.

Incident records are separate from the reusable troubleshooting guide. Case-specific facts, timelines, evidence, and RCA status belong here. Only generalized, sanitized lessons should be promoted into the reusable troubleshooting documentation.

## Incident Records

- [Ferrovax Incident](./ferrovax-incident/README.md)

## Handling Rules

```text
Keep each incident evidence boundary separate.
Do not import facts or conclusions from another incident unless the same evidence is provided again for the active incident.
Do not include client names, chat names, player names, raw account IDs, credentials, or unrelated personal information in reusable documentation.
```
